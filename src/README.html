\tBoxMe is OpenTTY Program that is developed focused in Virtual Machines experience in J2ME environment. It's important remember that API limitations and by some other factors (like Device Config.) running Virtual Machines in Dumbphones can impact the Device performance and can crash some times. [ Creating VM's ] \tYou can create a new Virual Machine by running command 'boxme new', or selecting this option in Main Menu of BoxMe. \tIt will ask some things to you about the Virtual Machine settings like a name for machine, name for the file, and the machine type. Currently is only supported Symbian, Native JVM to run JARs and an Android 2 simulation. [ Starting VM's]
